# TDD utilities

## tdd

```
    ./tdd "commit message"
```

Please modify this script to use your testing tool.

If the test pass, it executes `git add . && git commit -m message`

## tcr

```
    ./tcr "commit message"
```

Please modify this script to use your testing tool.

If the test pass, it executes `git add . && git commit -m message`
If the test fail, it executes `git reset --hard`
